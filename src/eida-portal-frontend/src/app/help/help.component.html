Map interaction

To change map zoom level use the +/- buttons located directly on the map or hold SHIFT key and use mouse/touchpad scroll.

To draw a dragbox hold SHIFT key and draw a rectangle on the map. The rectangle coordinates will be automatically filled in Events/Stations tabs.

Shortcut keys

Keys Result
1 Go to Events tab
2 Go to Stations tab
3 Go to Request tab
4 Go to Metrics tab
5 Go to Help tab
m m Toggle map
t t Toggle tooltips
